AddMoneyToWallet

UMLUseCase
Bet&Ruin :: Use Case Diagram :: AddMoneyToWallet

Description

Event flow

Main flow

  1. Admin gets in the AddFundsGUI
  2. System displays the AddFundsGUI
  3. Admin writes the ID of an existing User and indicates the amount of money to add
  4. Admin clicks on the AddFunds button
  5. System adds the indicated money to the indicated User

Alternative flow

  • X.A Admin can quit at any moment, END
  • 5.B Indicated User does not exist, display error message

Preconditions

  • Admin must be logged in
